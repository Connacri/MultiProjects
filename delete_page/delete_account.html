Suppression de votre compte

Si vous souhaitez supprimer définitivement votre compte et toutes vos données personnelles associées à l'application Check-it, veuillez remplir le formulaire ci-dessous. Vous recevrez un email de confirmation avant que la suppression soit effectuée.

Attention : Cette action est irréversible. Toutes vos données seront supprimées définitivement après confirmation.

Données supprimées

  • Informations de compte (email, nom, numéro de téléphone, mot de passe hashé)
  • Historique d'utilisation
  • Données personnelles stockées sur nos serveurs

Données conservées

Nous conservons certaines données minimales pendant 30 jours à des fins légales (lutte contre les fraudes ou obligations réglementaires), après quoi elles sont automatiquement supprimées.
