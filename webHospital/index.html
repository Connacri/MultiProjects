Questions fréquentes

Retrouvez les réponses aux questions les plus courantes

MediPlan Pro est-il compatible avec tous les systèmes d'exploitation ? +

Oui, MediPlan Pro est disponible pour Windows, macOS et Linux. Une version mobile est également disponible pour iOS et Android.

Puis-je essayer MediPlan Pro avant de l'acheter ? +

Absolument ! Notre version gratuite vous permet de tester les fonctionnalités de base avec jusqu'à 20 membres du personnel.

Comment se passe la migration depuis notre système actuel ? +

Notre équipe d'implémentation vous accompagne dans la migration de vos données depuis votre système actuel vers MediPlan Pro.

MediPlan Pro respecte-t-il les réglementations sur la protection des données ? +

Oui, MediPlan Pro est entièrement conforme au RGPD et aux réglementations en vigueur dans le secteur de la santé.
